a. Linus Torvalds fue el creador de Git, buscaba un sistema distribuido que pudiera usar en forma semejante a BitKeeper, el DVCS que usaba el proyecto del kernel de Linux para su control de versiones que dejó de ser gratuita en 2005, pero ninguno de los sistemas bajo software libre disponibles cumplía con sus requerimientos, especialmente en cuanto a desempeño. Y por eso creó su propia herramienta.
b. El propietario actual de Github es Microsoft, su compra se debió a que la compañía ha adoptado una estrategia nueva, mucho más abierta al mundo del software libre y a la integración con otras plataformas y servicios.
c. Se puede trabajar con Github directamente con Github Desktop, alternativamente a la terminal.
